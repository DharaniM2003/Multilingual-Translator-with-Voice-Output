# Multilingual Translator with Voice Output

## 📌 Project Overview
Multilingual Translator with Voice Output is a Python-based application that translates text into multiple languages and converts the translated text into speech. The system provides real-time translation, dynamic language selection, and Text-to-Speech (TTS) functionality through an interactive command-line interface.

---

## 🎯 Problem Statement
Language barriers often make communication difficult. This project aims to provide a simple and interactive translation tool that supports multiple languages and speaks the translated text, improving accessibility and multilingual communication.

---

## 📂 Features
- 🌍 Supports 30+ Languages
- 🔤 Real-Time Text Translation
- 🎙️ Text-to-Speech (TTS) Output
- 🌐 Dynamic Language Selection
- 🔁 Continuous Interactive Translation Loop
- ⚡ Fast and Lightweight Application

---

## 🛠 Technologies Used
- Python
- GoogleTrans
- pyttsx3
- Google Translate API

---

## 🚀 Project Workflow
1. Enter Input Text
2. Select Target Language
3. Translate Text
4. Display Translated Output
5. Convert Translation to Speech
6. Continue Interactive Translation
